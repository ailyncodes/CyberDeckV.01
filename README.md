<!-- GETTING STARTED -->
# Getting Started

## Items you will need
- x2 - M4*20
* x2 - M4 Nuts
+ Platic Glue
+ Raspberry Pi
+ 3.5 inch Mini Screen for Rasp Pi
+ (Optional) Foldable keyboard + mouse combo

## 3D Printer Settings
- Layer Resolution : 0.2 mm
* Filling : 100%

## Printing

1. Download STL files.
2. Print. 
3. GLUE 5 Pieces together.
4. Program and setup Raspberry Pi.
